
**基本 url： http://localhost:3000 **
# 1、注册接口文档

> 地址: '/users/register'

> 提交方式： post

> 参数： 用户名 username、密码 password、手机号 tel

> 返回结果： { code: '', message: ''}

> 返回参数说明：

|参数名称|备注|
|----|----|
|10000|该用户已注册，请直接登录|
|10101|注册成功|

# 2、登陆接口文档

> 地址: '/users/login'

> 提交方式： post

> 参数： 密码 password、手机号 tel

> 返回结果： { code: '', message: ''}

> 返回参数说明：

|参数名称|备注|
|----|----|
|10086|该用户未注册，请先注册|
|10010|登陆成功|
|10100|密码错误|
