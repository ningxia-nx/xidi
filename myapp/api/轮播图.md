**基本 url： http://localhost:3000 **
# 获取轮播图

> 地址: '/banner'

> 提交方式： get

> 参数： 

|参数|说明|默认值|
|-----|-----|----|
|type|类型|'home', 'kind', 'activity'|

> 返回结果： 

```
{
    "code": "200",
    "message": "获取轮播图数据成功",
    "data": [
        {
            "__v": 0,
            "bannerid": "banner_97da25a0-f965-11e9-be9d-69d5de3ce4f5",
            "type": "home",
            "img": "images/1.jpg",
            "href": ""
        }
    ]
}
```
